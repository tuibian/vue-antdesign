# 开箱即用

## 下载项目安装依赖
```
npm install
```

### 运行服务
```
npm run serve
```

### 运行mock本地数据
```
npm run mock
```

### 打包项目
```
npm run build
```

### 本框架特性
```
支持mockjs本地数据，支持国际化，支持换肤，支持token会话保持，支持侧边栏菜单后台配置。
```
